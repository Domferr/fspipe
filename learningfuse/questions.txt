- Supponiamo di avere due file, file1 e file2. Se apro file1 e poi apro file2 cosa succede?
- Due idee:
	- Se il client scrive sul file di nome "file1" allora il server deve leggere su un file con lo stesso nome
		- VANTAGGI: più comunicazioni tra lo stesso client e server
		- SVANTAGGI: devi ricordare lo stesso nome del file. Una soluzione può essere quella di contrattarla via handshake
	- Se il client scrive su un qualsiasi file, avviene l'invio al server che può leggere qualsiasi file
		- VANTAGGI: non devi sapere il nome del file e questo non deve essere identico
		- SVANTAGGI: non puoi avere più comunicazioni perchè le op su qualsiasi file consistono nella stessa operazione
